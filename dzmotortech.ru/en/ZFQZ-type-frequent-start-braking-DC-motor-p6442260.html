  • Description

ZFQZ type frequent start braking DC motor


Application Scenarios

Core applications include bridge/gantry cranes, screw-down devices of metallurgical rolling mills, elevator traction machines, mine winches, large water conservancy gate hoists, machine tool fast-moving mechanisms and other drive scenarios requiring frequent start-stop, forward and reverse rotation, and fast braking.

Product Certification

It has passed the ISO9001 Quality Management System certification, obtained the safety certification for special motors for hoisting machinery, and complies with GB/T 755 Rotating Electrical Machines – Rating and Performance and GB/T 21973 Technical Specification for Three-phase Asynchronous Motors for Hoisting and Metallurgy.
